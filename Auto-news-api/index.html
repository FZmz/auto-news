深耕近百年，

及时满足客户需求为成功之道

卡特彼勒作为一家在工程机械行业深耕近百年的全球性公司，始终致力于助力客户建设更加美好、更可持续的世界，通过我们在建设行业、资源行业、能源与交通这几大业务板块所提供的产品与服务解决方案，支持全球客户的业务发展，帮助客户取得成功。



及时满足客户所需是卡特彼勒的成功之道。今天，卡特彼勒超过190万台机械设备正在世界各地工作；我们的无人驾驶矿用卡车已实现超过40多亿公吨的零损工时伤害的搬运作业；我们已累计制造了2100多万台发动机；同时，数字化技术让全球超过120万台设备实现互联，为客户提供先进的服务。



无论在全球还是在中国，卡特彼勒一直通过整合创新的设备和服务助力客户迎接重大工程建设项目的挑战、满足他们阶段性的需求。其中不乏一些极具开创性和挑战性的、具有里程碑意义的项目，例如助力美国州际公路、巴拿马运河、阿波罗11号登月等全球重大项目；以及参与青藏铁路、三峡工程、西气东输、高铁等中国本土重大项目的建设。




更新全球企业战略，

将可持续性增加为战略关注

面对新时期的新挑战，卡特彼勒有哪些新作为？不久前，卡特彼勒更新了全球企业战略，将可持续性增加为战略关注之一，从之前的三项关注增加为四项。

卡特彼勒的可持续发展理念贯穿于我们与端到端价值链的代理商和供应商的合作之中。我们携手代理商全面开发和推出差异化的价值服务，如整机翻新服务，通过对设备发动机、零部件和系统的升级和翻新，使其开启新一轮的生命周期。这对于客户来说，不仅节约了成本，还是一种低碳的解决方案。




多样化的解决方案

助力客户实现低碳目标

当今的世界正在向提升能源效率、增加可持续能源使用、改进能源管理三个主要方向发展，以实现低碳的未来。对于卡特彼勒在各个行业的许多客户而言，能源转型是当下的“重大挑战”。卡特彼勒凭借着在创新和整合产品、系统和解决方案方面的专长，正在努力帮助客户实现这些目标。


卡特彼勒的客户分布在众多行业，客户的需求也是多种多样的。因此我们正努力提供多样化的解决方案以满足客户所需。



对于现有动力源，我们努力使燃料使用更灵活、更高效，例如提升低碳强度燃料的使用。


我们还将现有动力源与新技术匹配，例如使用集成了动力元件的电驱动传动装置，以及将可再生能源整合进入电力系统的微电网。


一些新技术已经可以替代现有动力源，如电池、能源存储，以及供固定与移动式应用使用的氢燃料电池。



助力客户实现低碳作业

卡特彼勒的大部分工程机械设备都可以使用诸如生物燃料或者氢化植物油此类低碳强度燃料。比如，英国的一家客户将它的整个车队的燃料从柴油替换为了氢化植物油，这帮助他们减少了高达85%的二氧化碳排放。


一家已有超过百年历史的承包商客户使用卡特彼勒最新的CAT®（卡特）D6 XE电动推土机运行约900小时后，确认在二氧化碳排放方面，可比其它型号的D6系列推土机减少35%。



与此同时，卡特彼勒非常了解客户对于全电驱动设备的需求，并在进行持续不断的探索，目前我们重点关注可以搭载电池或燃料电池的挖掘机和装载机产品。比如今年6月刚刚在中国推出的卡特彼勒旗下山工机械品牌的首款电驱装载机SEM658F，这款产品成功实现了尾气零排放。



除了工程机械行业，卡特彼勒的先进能源和动力源解决方案还已经在矿业、能源与交通等全球很多大型项目中投入广泛使用。目前，我们正在和全球客户展开合作，比如，我们正帮助必和必拓集团 (BHP)、力拓集团 (Rio Tinto) 等大客户打造全电驱动采矿机队和零排放矿场，这些采矿设备还可以搭载卡特彼勒的无人驾驶远程遥控技术。
